API#

日期类组件包括以下三种形式。

  • DatePicker

  • MonthPicker

  • RangePicker

注意： DatePicker、MonthPicker、RangePicker 部分 locale 是从 value 中读取，所以请先正确设置 moment 的 locale。

共同的 API#

以下 API 为 DatePicker、MonthPicker、RangePicker 共享的 API。

参数 说明 类型 默认值
allowClear 是否显示清除按钮 boolean true
disabled 禁用 boolean false
class 选择器 className string ''
size 输入框大小， large 高度为 32px， small 为 22px，默认是 28px string 无
locale 国际化配置 object 默认配置
disabledDate 不可选择的日期 (currentDate: moment) => boolean 无
openChange 弹出日历和关闭日历的事件 Event，参数值为 (boolean open) 无
placeholder 输入框提示文字 string -

DatePicker#

参数 说明 类型 默认值
value 日期 string, DatePicker，或者moment 无
format 展示的日期格式，配置参考moment.js string "YYYY-MM-DD"
change 时间发生变化的事件 Event，参数值为(date: moment) 无
showTimePicker 增加时间选择功能 boolean false
showToday 是否展示“今天”按钮 boolean true
disabledTime 不可选择的时间 function(date) => 对象，包含： /** 获取不可用的小时列表 */ disabledHours: () => number[]; /** 获取不可用的分钟列表 */ disabledMinutes: (hour: number) => number[]; /** 获取不可用的秒钟列表 */ disabledSeconds: (hour: number, minute: number) => number[]; 无

MonthPicker#

参数 说明 类型 默认值
format 展示的日期格式，配置参考 moment.js string "YYYY-MM"
change 时间发生变化的事件，发生在用户选择时间时 Event，参数值为(date: moment) -

RangePicker#

参数 说明 类型 默认值
value 日期 moment [] 无
format 展示的日期格式 string "YYYY-MM-DD HH:mm:ss"
change 时间发生变化的事件，发生在用户选择时间时 Event，参数值为(dates: moment, moment[]) 无
showTimePicker 增加时间选择功能 boolean false
disabledTime 不可选择的时间 function(dates: moment, moment , partial: 'start'|'end' )，返回值为对象，包含： /** 获取不可用的小时列表 */ disabledHours: () => number[]; /** 获取不可用的分钟列表 */ disabledMinutes: (hour: number) => number[]; /** 获取不可用的秒钟列表 */ disabledSeconds: (hour: number, minute: number) => number[]; 无
buttons 预设时间范围快捷按钮 object[]，例如：{{'{'}} text: '本月', start: moment(), end: moment.endOf('month') {{'}'}} 无
