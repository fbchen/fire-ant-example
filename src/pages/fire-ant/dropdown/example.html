Dropdown 下拉菜单

向下弹出的列表。

何时使用#

当页面上的操作命令过多时，用此组件可以收纳操作元素。点击或移入触点，会出现一个下拉菜单。可在列表中进行选择，并执行相应的命令。

菜单隐藏方式

默认点击菜单后将关闭菜单，如果设置了钩子函数menuClickHanlder，且该函数返回false，将停止关闭菜单。

API#

属性如下

参数 说明 类型 默认值
triggerAction 触发下拉的行为 Array<'click' | 'hover'> ['hover']
menuVisibleChange 菜单显示状态改变事件，参数为 visible: boolean Event -
dropdownPlacement 菜单弹出位置： bottomLeft bottomCenter bottomRight topLeft topCenter topRight String bottomLeft
menuClickHanlder 钩子函数: 默认点击菜单后将关闭菜单，如果此函数返回false，将停止关闭菜单。 Function -
dropdown 缩写写法，格式为：triggerAction? dropdownPlacement?。例如： hover bottomLeft hover click bottomLeft bottomLeft hover String -

菜单可由 ant-menu 取得，可监听select 事件，菜单还包括菜单项 ant-menu-item ，分割线 ant-divider 。

注意： ant-menu-item 必须设置唯一的 key 属性。

dropdown-button#

参数 说明 类型 默认值
type 按钮类型，和 Button 一致 string 'default'
trigger 触发下拉的行为 Array<'click' | 'hover'> ['hover']
overlay 菜单 ant-menu -
dropdownVisible 菜单是否显示 boolean -
popupVisibleChange 菜单显示状态改变事件，参数为 visible: boolean Event -
placement 菜单弹出位置： bottomLeft bottomCenter bottomRight topLeft topCenter topRight String bottomLeft
