API#

属性如下

成员 说明 类型 默认值
min 最小值 number undefined
max 最大值 number undefined
step 每次改变步数，可以为小数 number 1
precision 数值精度 number undefined
change 值变化事件，参数: (value: number) Event
disabled 禁用 boolean false
readOnly 是否只读。若为只读，则不可直接手工输入，只能上下调整。 boolean false
size 输入框大小。注：标准表单内的输入框大小限制为 `large`。可选 `large` `default` `small`。 string 无
formatter 指定输入框展示值的格式 function(value: number): string -
parser 指定从 formatter 里转换回数字的方式，和 formatter 搭配使用 function( string): number -
