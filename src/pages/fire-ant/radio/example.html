Radio 单选框

单选框。

何时使用#

  • 用于在多个备选项中选中单个状态。

  • 和 Select 的区别是，Radio 所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。

API#

Checkbox#

参数 说明 类型 默认值
checked 指定当前是否选中 boolean false
[(ngModel)] 传值 any -
change 值变化时事件，参数(value: any)。如果checked为true时，value传入Checkbox的value，否则传入空字符串。 Event -
check check变化时事件，参数(checked: boolean)。参数值说明：选中时返回true，否则返回false。 Event -
indeterminate 设置 indeterminate 状态，只负责样式控制 boolean false
