Tag 标签

进行标记和分类的小标签。

何时使用#

  • 用于标记事物的属性和维度。

  • 进行分类。

基本标签的用法，可以通过添加 closable 变为可关闭标签。可关闭标签具有 beforeClose close 两个事件。

动态添加和删除

用数组生成一组标签，可以动态添加和删除，通过监听删除动画结束的事件close实现。

多彩标签

我们添加了多种预设色彩的标签样式，用作不同场景使用。如果预设值不能满足你的需求，可以设置为具体的色值。

可选择

可通过 CheckableTag 实现类似 Checkbox 的效果，点击切换选中效果。

该组件为完全受控组件，不支持非受控用法。

API#

Tag#

参数 说明 类型 默认值
color 标签色 string -
closable 标签是否可以关闭 boolean false
beforeClose 关闭前事件，参数：(event: Event)。若调用event.preventDefault();可以阻止关闭。 Event -
close 关闭后事件，参数：(event: Event)。 Event -

Tag.CheckableTag#

参数 说明 类型 默认值
checked 设置标签的选中状态 boolean false
change 点击标签时触发的事件，参数：(checked: boolean) Event -
