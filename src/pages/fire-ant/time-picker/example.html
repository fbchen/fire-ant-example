API#

参数 说明 类型 默认值
value 当前时间 moment 无
openChange 面板打开/关闭时的时间 Event，参数值为：(open: boolean) 无
placeholder 没有值的时候显示的内容 string "请选择时间"
change 时间发生变化的事件 Event，参数值为：(time: moment) 无
format 展示的时间格式 string "HH:mm:ss"
disabled 禁用全部操作 boolean false
disabledHours 禁止选择部分小时选项 /** 获取不可用的小时列表 */ disabledHours: () => number[]; 无
disabledMinutes 禁止选择部分分钟选项 /** 获取不可用的分钟列表 */ disabledMinutes: (selectedHour: number) => number[]; 无
disabledSeconds 禁止选择部分秒选项 /** 获取不可用的秒钟列表 */ disabledSeconds: (selectedHour: number, selectedMinute: number) => number[]; 无
hideDisabledOptions 隐藏禁止选择的选项 boolean false
class 选择器类名 string ''
dropdownClass 弹出层类名 string ''
